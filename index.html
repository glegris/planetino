Planetino

Planetino is a 3D engine for J2ME/MIDP2 devices. It focus on First Person Shooting games.

Planetino was initially developed as a side project of MIDPath in order to test 3D features.

Old project site: https://code.google.com/p/planetino/

Features:

  • Physics
  • Easy map creation
  • Collision Detection
  • A.I
  • Pure Java renderer

Coming soon:

  • JSR184/M3G rendering
  • Networking
